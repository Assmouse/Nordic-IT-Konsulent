## Formål

Formålet med projektet er at udvikle en simpel hjemmeside for den fiktive virksomhed Nordic IT Consulting.

Hjemmesiden skal præsentere virksomhedens services, medarbejdere, projekter, karrieremuligheder og kontaktoplysninger.

## Målgruppe

Målgruppen er potentielle kunder, som søger IT-konsulentydelser, samt potentielle medarbejdere med interesse for virksomheden.

## Projektmål

Projektets mål er at:

- Udvikle en hjemmeside bestående af minimum 8 HTML-sider.
- Implementere responsivt design ved hjælp af CSS.
- Anvende Git og GitHub til versionsstyring.
- Demonstrere anvendelse af softwareudviklingsmetoder fra kurset.

## Teknologier

Projektet er udviklet ved hjælp af:

- HTML5
- CSS3
- GitHub
- Visual Studio Code

## Udviklingsmetode

Projektet er udviklet inkrementelt med inspiration fra Sommervilles softwareudviklingsproces.

Udviklingen er gennemført ved hjælp af branches, commits og merges i GitHub.
