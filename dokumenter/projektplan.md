## Formål

Formålet med projektet er at udvikle en hjemmeside for den fiktive virksomhed Nordic IT Consulting.

## Aktiviteter

| Aktivitet                       | Status       |
| ------------------------------- | ------------ |
| Oprettelse af GitHub repository | Gennemført   |
| Oprettelse af branches          | Gennemført   |
| Udvikling af HTML-sider         | Gennemført   |
| Udvikling af CSS-design         | Gennemført   |
| Test af navigation              | Gennemført   |
| Udarbejdelse af dokumentation   | Igangværende |
| Udarbejdelse af UML-diagrammer  | Igangværende |
| Endelig aflevering              | Planlagt     |

## Udviklingsmetode

Projektet er gennemført ved hjælp af en inkrementel udviklingsproces, hvor funktionalitet er blevet implementeret og testet løbende.

Versionsstyring er håndteret gennem GitHub ved brug af branches, commits og merges.
