## Systembeskrivelse

Nordic IT Consulting er en statisk hjemmeside udviklet som en del af projektet. Formålet med systemet er at præsentere et fiktivt IT-konsulentfirma og dets services.

## Teknologier

Projektet er udviklet ved hjælp af følgende teknologier:

* HTML5
* CSS3
* JavaScript
* GitHub

## Filstruktur

Projektet er organiseret i følgende mapper:

* css
* sider
* dokumenter

HTML-siderne er placeret i mappen *sider*, mens stylesheets er placeret i mappen *css*.

## Navigation

Hjemmesiden benytter en fælles navigationsmenu på alle sider. Navigationen er implementeret ved hjælp af HTML-links mellem de forskellige sider.

## Responsive Design

Der er implementeret et grundlæggende responsivt design ved hjælp af CSS media queries.

Ved skærmstørrelser under 768 pixels vises navigationsmenuen lodret for at forbedre brugervenligheden på mindre enheder.

## Kontaktformular

Kontaktsiden indeholder en kontaktformular, hvor brugeren kan indtaste navn, e-mail og besked.

Der er implementeret JavaScript, som viser en bekræftelsesbesked ved afsendelse af formularen. Systemet indeholder ikke en backend-løsning, og data gemmes derfor ikke i den nuværende version.

## Versionsstyring

Projektet er udviklet ved hjælp af GitHub.

Udviklingsprocessen har omfattet:

* Branches
* Commits
* Pull Requests
* Merges

Dette har gjort det muligt at styre ændringer og dokumentere udviklingsforløbet.

## Kørsel af Systemet

Systemet kan afvikles ved at åbne index.html i en webbrowser.

Under udviklingen er Visual Studio Code og Live Server anvendt til lokal test af hjemmesiden.
